# GoRyDz - Car Rental Website

A website where people can rent cars online.

## What it does

- Show available cars with pictures and prices
- Book a car for specific dates
- Chat with AI bot for help
- Admin can manage cars and bookings

## Built with

- Django (Python web framework)
- HTML, CSS, JavaScript
- SQLite database
- Google Gemini AI for chatbot

## How to run

1. Download the code
2. Install Django: `pip install django`
3. Run: `python manage.py runserver`
4. Open browser: `http://127.0.0.1:8000/`

## Pages

- Home page - welcome screen
- Cars page - see all cars
- Booking page - rent a car
- Chat page - ask questions to AI
- Contact page - get in touch
